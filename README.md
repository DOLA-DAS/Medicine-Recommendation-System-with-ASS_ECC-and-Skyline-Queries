This project processes raw medical prescription data (OCR output from prescriptions), encrypts sensitive medicine details using ElGamal homomorphic encryption and Shamir's Secret Sharing, and recommends medicines for given symptoms using skyline query algorithms.

**Features-**
Data Extraction,  
Secret Sharing, 
ElGamal Encryption, 
Homomorphic Multiplication, 
MFA / OTP Authentication,
Skyline Queries-Block Nested Loop (BNL), Divide and Conquer, Dominance-Based Frequency Ranking, 
Jaccard Similarity — Compares overlap between the three skyline methods
